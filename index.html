Aegis Backend

Secure and Scalable Django API

📋 About This Project

This is a Django REST Framework backend API with:

  • JWT Authentication
  • MySQL/SQLite Database
  • Chatbot Integration
  • RESTful API Endpoints

✅ How to Run Locally

cd E:\aegis_backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

Then open: http://localhost:8000

⚠️ GitHub Pages Limitation

This is a Django backend that requires a Python server to run.

To deploy this project online, use:

  • Render.com (free Django hosting)
  • Heroku (free tier)
